# unquitube-iisoft
  Proyecto de aplicacion web para reproduccion de videos en real time entre diferentes usuarios.

## Tecnologias:
-- Backend

---- Lenguaje: Java

---- Framework: Spring

---- BD: MySQL

---- Server: TomCat

-- Frontend
---- Lenguaje: JavaScript, Css, Html
---- Framework: AngularJS
---- Librerias: Bootstrap
---- Server: Node
